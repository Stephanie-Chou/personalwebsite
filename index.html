FlipSide

 Dev Bootcamp final 8 day project. Web application news feed which brings together differing sides of the story. Uses Google News scraper and the Alchemy API for natural language processing. Wrote a program which compared scraped news articles based on their keywords and sentiment and displayed the maximally dissimilar articles as pairs, giving users multiple perspectives on the major issues of the day. On our frontend, we used a combination of React and D3.js to create a interactive user interface and visualize user data. source code
